Описание

Заготовки по одной подаются из магазина подачи присосками, работа присосок синхронизирована с главной передающей цепью. Далее посредством толкателей заготовки перемещаются к узлу формирования.

В процессе горизонтального перемещения заготовок, происходит нанесение клея горячего плавления в требуемых местах. Затем в узле формирования заготовка приобретает форму лотка при помощи головы формирователя, происходит загиб верхних клапанов при помощи механических устройств.
